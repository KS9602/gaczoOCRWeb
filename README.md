Aplikacja w trakcie rozwoju.

GaczoOCRWeb jest aplikacją przeglądarkową do zczytywania danych z faktur. Dodawane są nowe kolumny z wyliczonymi parametrami i dokument jest zwracany jako .xlsx spakowany do zip.
Aplikacja zbudowana jest w architekturze mikroserwisowej. Wykorzystuje spring'a, kafkę, reacta (glównie wygenerowany w ai jako prosty frontend), keycloaka.
